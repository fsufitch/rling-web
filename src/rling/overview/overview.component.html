Rifterlings Starter Overview Pack

These exported overview settings are basic settings for operating as part of Rifterlings in the Gallente Militia. They contain markings for both ally status, and engagement status (suspect flagging, limited engagement, etc). It also contains a basic set of preset tabs useful for lowsec PvP.

This is supposed to be a "starter pack". If you already have an ultra-customized overview, this is likely not an improvement in your case.
